# 에듀워치 FE

## 초, 중, 고등학생을 위한 온라인 관리형 독서실, 에듀워치입니다.

주요 기능은 다음과 같습니다. 
- 화상통화를 이용한 캠스터디
 : 대학생 멘토가 실시간 화상 회의 모니터링을 하며 학습 여부를 체크합니다.
- 학부모의 자녀 학습 모니터링 
: 입/퇴실 여부, 실시간 화상 모니터링, 학습 보고서 전달 기능을 통해 학부모가 자녀의 학습 상태를 확인할 수 있습니다. 
- 대학생 멘토의 질의응답 및 학습 관리 
: 멘토의 실시간 화상 모니터링을 통해, 집중력 유지 및 자유로운 질의응답이 가능합니다.
